Introduction Code Playground

Kids Code Playground

The kids coding playground is combines physical play with digital learning, fostering creativity and collaboration among young learners.

Kids can use Scratch, a kid-friendly block-based programming language, to design animations, create games, and tell interactive stories. The station features tutorials and challenges that help kids understand the basics of coding in a visual way.

Game Development Zone.

Tools and resources for kids to create their own video games, including character design, coding mechanics, and storytelling.

Kids can brainstorm and create their own games, either individually or in teams. They will learn critical thinking and problem-solving skills as they navigate the game development process from concept to execution.
